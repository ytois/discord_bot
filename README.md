# Discord Bot

Discord Bot for Glitch
https://discord-zoo-bot.glitch.me

## ENV

| name              | value                    |
| ----------------- | ------------------------ |
| NODE_ENV          | production / development |
| DISCORD_BOT_TOKEN | Discord bot token        |
| PORT              | express listen port[^1]  |

[^1]: Glitch で使う場合は設定不要。設定した場合 3000 以外はエラーになる。
